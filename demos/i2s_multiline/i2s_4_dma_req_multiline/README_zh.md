# I2S多通道DMA传输示例

## 概述

- 该实例工程展示了使用I2S接口进行多通道DMA传输的功能

## 限制要求

- I2S的每条line都支持独立的DMA请求(如HPM6P00系列)

## 工作原理

在I2S4通道发送模式下：
- 系统使用4个独立的DMA通道
- 每个DMA通道专门响应一个I2S Line的DMA发送请求
- DMA控制器将数据源自动搬运到对应I2S Line的TXD寄存器中

## 运行要求

- 需要根据实际硬件配置I2S引脚，并使用逻辑分析仪等工具观察引脚波形
